# Paint Interactive

A simple node.js server for an interactive installation piece.

## Demo
![Paint Interactive Demo](demo/interactive-paint-demo.gif)

## Getting Started

1.  install node.js
2.  npm install
3.  npm start


## Prerequisites

Requirements are Node 10.16.0 and npm 6.9.0


## Deployment

Local machine
