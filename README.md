# Paint Interactive

The server-client side of an interactive installation piece.


## Getting Started

1.  install node.js
2.  npm install
3.  npm start


## Prerequisites

Requirements are Node 10.16.0 and npm 6.9.0


## Deployment

Local machine

* [npm](https://github.com/npm/cli)
